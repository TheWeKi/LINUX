# Setup

    1 Create bootable drive with arch iso
    2 Enter the installation media
    3 Create Partitions
        3.1 BOOT    1G
        3.2 SWAP    16G
        3.3 ROOT    REMAINING
    4 Install git and Clone the repository
    5 In repo, Make all scripts executable
    6 run ./archinstall.sh
